Compétence 1

Développer une application informatique

Description

Cette compétence vise à maîtriser le développement d'applications informatiques en utilisant différents langages de programmation et paradigmes de développement.

Activités

Analyse et conception

Analyser les besoins, concevoir l'architecture logicielle.

Implémentation

Coder l'application en respectant les bonnes pratiques.

Tests et validation

Tester et valider le fonctionnement de l'application.

Documentation

Rédiger la documentation technique et utilisateur.

Modules associés

  • Module 1: Introduction à la programmation
  • Module 2: Structures de données
  • Module 3: Programmation orientée objet
  • Module 4: Bases de données
  • Module 5: Développement web

SAE (Situations d'Apprentissage et d'Évaluation)

Les SAE sont des projets pratiques qui permettent de mettre en œuvre les compétences acquises.

  • SAE 1: Développement d'une application simple
  • SAE 2: Gestion de base de données
  • SAE 3: Projet web
